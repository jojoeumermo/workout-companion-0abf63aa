11. IA NO CELULAR — RESOLVER DIREITO
A IA funciona no PC, mas não funciona no celular. Isso não é aceitável.

Quero resolver isso da forma correta.

Regras:
- não chamar a API de IA diretamente no frontend/mobile
- implementar via backend
- usar rota segura no backend para intermediar as chamadas
- proteger chave da API
- garantir funcionamento no celular e no PC
- evitar problemas de CORS, exposição de chave e inconsistência entre web e app mobile

Quero que você:
- revise a arquitetura atual da IA
- mova a integração para backend se ainda estiver no frontend
- faça a solução funcionar de forma estável tanto no celular quanto no desktop
- teste o fluxo completo
- se não for possível deixar estável agora, remover a IA e qualquer UI quebrada relacionada
